## Turning it off

### Chrome

`Preferences` > `Extensions` > Uncheck `Enable`

### Firefox

`Tools` > `Add-ons` > `Extensions` > Click `Disable`

### Safari

`Preferences` > `Extensions` > Uncheck checkbox next to this plugin
